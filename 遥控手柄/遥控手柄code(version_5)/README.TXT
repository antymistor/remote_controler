卷扬机控制系列

对于卷扬机A和卷扬机B,通过字母A,B作为区分，下面以卷扬机A为例介绍，卷扬机B同理。

A0+x1x2x3 :A以x1x2x3的速度正转
A1+x1x2x3 :A以x1x2x3的速度反转
A2000     :A停转
A3+x1+x2x3:A以x1的启动速度转x2x3的时间正转至停
A4+x1+x2x3:A以x1的启动速度转x2x3的时间反转至停

对于机器人主体其接受地址为C
C0+x1x2x3  :控制机器人主体上拉x1x2x3mm
C1+x1x2x3  :控制机器人主体下放x1x2x3mm
C2000      :刷新计步器标记并停转一切
C3000      :使其发送当前计步器结果
C4+0/1+x1x2:使摇臂手动控制左/右以x1x2的速度移动
C5+0/1+x1x2:使摇臂自动控制左/右以x1x2的速度移动
C6+0/1+x1x2:使机器人主体横向向左/右以x1x2的速度移动
C7+0/1+x1x2:使机器人主体顺/逆时针以x1x2的转速旋转

阀控开关组的接受地址为D
D0000,D0001,D1000,D1001:阀0/1 开/关
